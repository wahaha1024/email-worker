# Email-Worker 完整功能演示指南

## 🎉 系统概述

Email-Worker 是一个部署在 Cloudflare Workers 上的现代化邮件+RSS 订阅管理系统，采用精美的 Koobai 设计风格。

**访问地址**: https://email.zjyyy.top

---

## 📱 界面导览

### 底部导航栏
固定在底部的磨砂玻璃导航栏，包含 6 个功能按钮：

```
[收件箱] [日志] [订阅] | [筛选] [搜索] [编辑]
```

- **左侧**：页面导航
  - 收件箱：查看邮件和 RSS 文章
  - 日志：系统处理日志
  - 订阅：RSS 订阅管理

- **右侧**：工具菜单
  - 筛选：按分类筛选内容
  - 搜索：全文搜索
  - 编辑：批量操作

## 🔥 核心功能演示

### 1️⃣ 合并视图（首页）

**访问**: https://email.zjyyy.top

**特点**：
- 邮件和 RSS 文章统一时间线
- 按发布时间倒序排列
- 清晰的类型标识：
  - 📧 = 邮件
  - 📰 = RSS 文章
  - 🌐 = RSS 来源标记

**类型筛选栏**：
```
[全部] [邮件] [RSS]
```
点击切换显示内容类型

---

### 2️⃣ 筛选功能演示

**操作步骤**：
1. 点击底部"筛选"按钮
2. 弹出筛选菜单
3. 选择筛选条件：
   - 全部
   - 收件箱
   - 重要
   - 未读

**效果**：
- 自动跳转到筛选结果页
- 保持当前内容类型（全部/邮件/RSS）
- URL 示例：`/?type=all&is_read=0`

**菜单样式**：
```
┌─────────────┐
│ 📥 全部     │
│ ✉️  收件箱   │
│ ⭐ 重要     │
│ ⭕ 未读     │ ← 高亮当前项
└─────────────┘
```

---

### 3️⃣ 搜索功能演示

**操作步骤**：
1. 点击底部"搜索"按钮
2. 顶部弹出搜索框
3. 输入关键词（如："技术"）
4. 按回车或点击"搜索"

**搜索范围**：
- 邮件标题和内容
- RSS 文章标题和摘要
- 同时搜索两种类型

**效果**：
```
┌──────────────────────────────┐
│ 🔍 [技术_____________] [搜索] │
└──────────────────────────────┘
```

**搜索结果**：
- 高亮匹配项
- 保持类型筛选状态
- 显示匹配的邮件和文章

---

### 4️⃣ 编辑功能演示

**完整流程**：

#### 步骤 1：进入选择模式
1. 点击底部"编辑"按钮
2. 点击"选择内容"
3. 进入选择模式

**效果**：
- 每个卡片左侧显示复选框
- 点击卡片等同于勾选
- "选择内容"变为"退出选择"

#### 步骤 2：选择项目
- 勾选任意邮件和 RSS 文章
- 支持混合选择
- 选中项带复选标记

**示例**：
```
☑️ 📧 彭博财经早茶...
☑️ 📰 科技爱好者周刊...
☐ 📧 Gmail 转发确认...
☑️ 📰 iPad 绘画工具...
```

#### 步骤 3：批量操作
再次点击"编辑"，选择操作：

**标记已读**：
- 将选中的邮件和文章标记为已读
- 自动调用对应 API
- 刷新页面显示结果

**删除**：
- 批量软删除选中项
- 需要确认
- 同时处理邮件和 RSS

---

### 5️⃣ RSS 订阅管理

**访问**: https://email.zjyyy.top/feeds

**功能**：

#### 添加订阅源
1. 点击"➕ 添加订阅源"
2. 填写信息：
   ```
   订阅源名称: 阮一峰的网络日志
   RSS 地址: https://www.ruanyifeng.com/blog/atom.xml
   分类: [技术] [新闻] [博客] [其他]
   Cron 表达式: 0 */6 * * *
   ```
3. 选择或自定义 Cron 表达式：
   - 每小时
   - 每6小时 ← 推荐
   - 每天
   - 自定义

4. 点击"添加订阅"
5. 自动抓取首批文章

#### 管理订阅源
每个订阅源显示：
```
🌐 阮一峰的网络日志        [✏️] [🔄] [🗑️]
https://www.ruanyifeng.com/blog/atom.xml
─────────────────────────────────────────
📅 最后抓取：30分钟前
📊 Cron: 0 */6 * * *
✅ 状态正常
```

**操作按钮**：
- ✏️ 编辑：修改配置
- 🔄 刷新：手动抓取
- 🗑️ 删除：移除订阅源

## 🚀 使用场景示例

### 场景 1：查看今天的新闻
1. 访问首页
2. 点击"筛选" → "未读"
3. 查看所有未读邮件和文章
4. 点击感兴趣的内容查看详情

### 场景 2：搜索技术文章
1. 点击"搜索"
2. 输入"React"
3. 查看所有包含 React 的邮件和文章
4. 点击类型筛选切换到"RSS"仅看文章

1. 点击"编辑" → "选择内容"
2. 勾选多封已读邮件
3. 点击"编辑" → "删除"
4. 确认删除

### 场景 4：添加新的 RSS 源
1. 访问 `/feeds`
2. 点击"添加订阅源"
3. 填写信息并设置每6小时抓取
4. 等待自动抓取首批文章
5. 返回首页查看新文章

## 💡 使用技巧

### 技巧 1：组合筛选
URL 参数可以组合：
```
/?type=rss&category=tech&search=AI
```
显示包含"AI"的技术类 RSS 文章

### 技巧 2：RSS 抓取频率优化
根据更新频率设置 Cron：
- 高频更新（新闻）：`*/30 * * * *` (每30分钟)
- 中频更新（科技博客）：`0 */6 * * *` (每6小时)
- 低频更新（个人博客）：`0 8 * * *` (每天早8点)

### 技巧 3：搜索技巧
- 搜索会匹配标题和内容
- 搜索不区分大小写
- 支持中英文混合搜索

### 技巧 4：批量操作加速
1. 进入选择模式
2. 直接点击卡片即可选择（无需点击复选框）
3. 快速勾选多个
4. 执行批量操作
